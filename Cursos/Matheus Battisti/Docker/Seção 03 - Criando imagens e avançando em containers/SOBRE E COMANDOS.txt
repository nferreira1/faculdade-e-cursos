COMANDOS


BAIXAR IMAGENS SEM EXECUTAR
docker pull NOME_DA_IMAGEM

LISTAR IMAGENS QUE ESTÃO INSTALADAS
docker image ls

BUILDAR COM O DOCKER
docker build CAMINHO

BUILDAR E RENOMEAR A IMAGEM
docker build -t NOME_IMAGEM:TAG_IMAGEM CAMINHO

REMOVER IMAGENS E CONTAINERS NÃO UTILIZADOS
docker system prune

REMOVER CONTAINER AUTOMATICAMENTE APÓS UTILIZAR
docker run --rm NOME_DO_CONTAINER

COPIAR ARQUIVOS ENTRE CONTAINERS
docker cp NOME_DO_CONTAINER:CAMINHO_DO_ARQUIVO_QUE_DESEJA_COPIAR CAMINHO_PARA_ONDE_O_ARQUIVO_VAI

VERIFICAR INFORMAÇÕES DE PROCESSAMENTO
docker top NOME_DO_CONTAINER

VERIFICAR INFORMAÇÕES DE PROCESSAMENTO (INFORMAÇÕES GERAIS) DE TODOS OS CONTAINERS
docker stats

VERIFICAR DADOS DE UM CONTAINER
docker inspect NOME_DO_CONTAINER
